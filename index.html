ng-clamper

A module for cross-browser line clamping.

Download this project as a .zip file Download this project as a tar.gz file

About this module

ng-clamper adds cross-browser support for line clamping to your angular app.

Usage

To use ng-clamper you'll need to add the 'ng-clamper' class to the element you'd like clamped.
